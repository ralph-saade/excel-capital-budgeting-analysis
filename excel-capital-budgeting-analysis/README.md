This project is an Excel-based capital budgeting model that compares three investment projects using NPV, IRR, and Payback Period. It includes:
Automated calculations for financial metrics.
Cumulative cash flows and project comparison.
Dashboards with charts and conditional formatting to visualize project performance.
Insights for investment decision-making.

Skills / Tools Used:
Excel (Formulas, Conditional Formatting, Charts, Data Analysis)
Financial Modeling (NPV, IRR, Payback Period, Capital Budgeting)
Data Visualization

How to Use / View:
Open the Excel file in Microsoft Excel (2016 or later recommended).
Navigate through the “Calculations” and “Cash Flow” sheets.
Review charts, KPIs, and calculated metrics for this project.
